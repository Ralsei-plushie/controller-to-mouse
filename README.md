# Controller To Mouse
This simple Python script lets you control your mouse with a controller!

### **Controls**:
* D-pad left - Moves mouse left.
* D-pad right - Moves mouse right.
* D-pad up - Moves mouse up.
* D-pad down - Moves mouse down.
* R1 - Right clicks.
* L1 - Left clicks.

### Failsafe:
- Using this Python script, if you need a way to turn it off but don't have access to the terminal, put your cursor in the top left corner of screen!
- It will trigger a fail-safe just incase.

# Note:
This was tested only with a PS4 controller, not guaranteed to work on all controllers.
